Horry County Schools Technology Fair

For my Sophomore year of tech fair, William Atkinson and I presented a small project based on LUA coding; it was a game that showed the flexibility and functionality of LUA code in such a small computer. We used an app called TIC-80, which ran the code. TIC-80 is an app that can be used to create a limited computer environment to create minigames. Its max supported file size is 64kb, which leaves the user with a completely minimal workspace. We won first place in Gaming and App Design.
